# git 으로 파일 올리는 법
1. git bash 실행
2. repository 폴더로 이동
3. git remote add origin [깃허브주소]
  origin 이라는 별명을 지어주고, 원격의 깃허브랑 지금 본인의 개발환경이랑 연결해주는 명령어
4. git remote : 중간확인(origin으로 되었나)
5. 파일 및 폴더 올림.
6. git add .
7. git status
8. git commit -m "커밋메시지"
9. git push origin main : main은 브랜치명이다.
10. 내용이 바뀌었는지 확인
